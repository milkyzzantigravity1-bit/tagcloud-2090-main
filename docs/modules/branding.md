# Модуль `branding`

Визуальная идентичность ГБОУ Школа №2090: лого, цвета, типографика, layout. Тонкий «модуль» — это не код, а правила.

## Файлы

- `static/logo2090.png` — официальный логотип школы
- `src/app.css` — CSS-переменные с палитрой
- `src/lib/theme.ts` — те же цвета как TS-константы (для wordcloud, для inline-стилей)
- `src/routes/+layout.svelte` — глобальный layout с шапкой/футером

## Палитра (производная от лого)

| Имя | Hex | Использование |
|---|---|---|
| `--c-navy` | `#0E2A5C` | основной текст брендовых элементов, кнопки, шапка |
| `--c-blue` | `#2D9FDA` | акцент (links, highlights) |
| `--c-gold` | `#E0B73A` | вторичный акцент (CTA, бейджи) |
| `--c-bg` | `#FFFFFF` | фон |
| `--c-surface` | `#F7F8FA` | вторичный фон (карточки) |
| `--c-text` | `#1A1A1A` | основной текст |
| `--c-muted` | `#6B7280` | вторичный текст |
| `--c-border` | `#E5E7EB` | бордеры |
| `--c-danger` | `#DC2626` | ошибки |

## Типографика

- Шрифт: `Inter` (latin + cyrillic, бесплатный, через `<link>` в `app.html` — этап 6) с системным fallback
- Базовый размер: `16px`, `line-height: 1.5`
- Заголовки: `font-weight: 600`, `letter-spacing: -0.02em`, цвет `--c-navy`

## Layout

- Контейнер: `max-width: 880px`, центрирование, `padding: 48px 24px`
- Шапка: 1 строка, лого + название "Облако тегов"
- Футер: 1 строка, цвет `--c-muted`

## Радиусы и тени

- `--radius: 8px` (кнопки, карточки)
- `--radius-lg: 12px` (большие блоки)
- `--shadow-sm: 0 1px 2px rgba(14, 42, 92, 0.06)` (тонкие)
- `--shadow: 0 4px 12px rgba(14, 42, 92, 0.08)` (модалки, фокус)

## Минимализм — правила

- ОДНА колонка контента, центрирование. Не двух- и не трёхколоночные layouts.
- Не использовать градиенты, кроме самых тонких (на CTA).
- Кнопки: solid navy (primary), white с border (secondary). Без gradients/shadows.
- Иконки: emoji или Lucide SVG, не более 1-2 на экран.
- Анимации: только `transition: 0.15s ease` на hover/focus. Никаких сложных motion.

## Готчи

- НЕ менять hex-цвета произвольно. Если нужен новый оттенок — обсудить и добавить в палитру.
- НЕ использовать сторонние UI-киты (Bootstrap, Material) — они принесут свои стили и сломают минимализм.
- НЕ ставить лого на фон, отличный от белого, без проверки контраста.
- Лого `static/logo2090.png` — НЕ редактировать (брендбук школы). Если нужен SVG-вариант — попросить отдельный файл.
- Все цвета — через CSS-переменные, НЕ хардкодить hex в компонентах. Для wordcloud — через `palette` из `theme.ts`.
- На этапе 6 добавить favicon (вырезать центральную часть лого 32×32) и OG-image (1200×630 с лого + "Облако тегов").
